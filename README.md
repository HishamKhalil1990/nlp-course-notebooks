## Instructions

1. Create a virtual envrionment

   ```bash
	python3 -m venv venv
   ```

2. Activate the environment and install the required packages
   ```bash
	source venv/bin/activate
	pip3 install -r requirements.txt
   ```

3. Run Jupyter Notebook sever
	```bash
	jupyter notebook
	```
